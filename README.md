# desafiofinal
Objetivo Simular um fluxo básico de trabalho em equipe usando Git, incluindo criação de branches, commits e resolução de conflitos. Utilizando tudo que aprendemos durante as aulas, é hora de colocar a mão na massa:

 
1. Inicialize um repositório, adicione um arquivo .txt a branch master/main e deixe ele remoto
2. Crie uma nova branch, adicione um arquivo .txt e faça um commit
3. Dê um push nas atualizações da sua nova branch deixando ela remota
4. Volte para a branch master/main e faça um merge com a branch que você criou no passo 2
5. Commit as alterações e faça um push para deixar tudo atualizado no repositorio remoto
6. Desafio Extra (🔥 Conflito)
Na branch master/main, edite o arquivo .txt adicionando uma descrição
Faça commit
Vá para a branch criada no passo 2
Edite a mesma linha do .txt com outro conteúdo
Faça commit
Tente fazer merge dessa branch na master/main
Resolva o conflito, dê o commit e o push
